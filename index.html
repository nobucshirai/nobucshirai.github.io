生成AI活用ツール一覧

以下は、このリポジトリで利用可能なツールの一覧です。各ツールの説明とリンクが記載されています。

会議要約用プロンプトテンプレート

このツールは、会議の参加者一覧とトランスクリプトを基に、会議要約用のプロンプトを生成するテンプレートを提供します。概要とネクストアクションの提示をサポートします。

翻訳プロンプト（日本語→英語）

このツールは、日本語の文章を英語に翻訳するためのプロンプトを生成します。直訳から始め、段階的に自然な英語表現へと改善します。

Natural Phrasing Guide

This tool provides suggestions for more natural phrasing in conversation transcripts, helping improve clarity and flow.

CLI Tool Prompt Generator

This tool helps you generate command-line interface prompts for creating Python3 scripts with various specifications.

Sentence Refinement

This tool reviews and refines sentences through iterative steps, providing suggestions to improve clarity and style.

Prompt Template Parser

This tool converts extended Markdown with custom syntax into complete HTML prompt templates. It supports inline inputs, checkboxes, and file uploads.

フランス語単語クイズ

このツールは、フランス語の単語をクイズ形式で学習するためのツールです。正しいフランス語の単語を選択することで、語彙力の向上をサポートします。
