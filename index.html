イタリア語ボキャブラリークイズ

  • イタリア語「基本動詞」単語クイズ1

    イタリア語の基本動詞 essere / avere の活用をクイズで学習できます。音声で活用形を確認しましょう。

  • イタリア語「基本動詞」単語クイズ2

    イタリア語の基本動詞 andare / venire の活用をクイズで学習できます。音声で活用形を確認しましょう。

  • イタリア語「基本動詞」単語クイズ3

    イタリア語の基本動詞 volere / dire の活用をクイズで学習できます。音声で活用形を確認しましょう。

  • イタリア語「挨拶」単語クイズ

    あいさつフレーズを中心にしたイタリア語語彙クイズです。旅行前の練習に最適です。

  • イタリア語「数字1–10」単語クイズ

    イタリア語の数字 1〜10 をクイズ形式で覚えましょう。

  • イタリア語「数字11–20」単語クイズ

    数字 11〜20 を扱うイタリア語クイズです。さらに数を広げて覚えましょう。

フランス語ボキャブラリークイズ

  • フランス語「基本動詞」単語クイズ1

    フランス語の基本動詞 être / avoir の活用をクイズ形式で学習できます。音声付でリスニング練習にも最適です。

  • フランス語「基本動詞」単語クイズ2

    フランス語の基本動詞 aller / venir の活用をクイズ形式で学習できます。音声付でリスニング練習にも最適です。

  • フランス語「基本動詞」単語クイズ3

    フランス語の基本動詞 vouloir / dire の活用をクイズ形式で学習できます。音声付でリスニング練習にも最適です。

  • フランス語「挨拶」単語クイズ

    挨拶表現を中心に学べるフランス語語彙クイズです。発音音声でフレーズを確認しましょう。

  • フランス語「数字1–10」単語クイズ

    数字 1〜10 の読み方と綴りを覚えるためのフランス語クイズです。

  • フランス語「数字11–20」単語クイズ

    数字 11〜20 を対象としたフランス語語彙クイズ。1–10 をマスターしたら次のステップへ！

その他ツール

  • 会議要約用プロンプトテンプレート

    会議の参加者一覧とトランスクリプトを基に、会議要約用のプロンプトを生成するテンプレートを提供します。

  • 翻訳プロンプト（日本語→英語）

    日本語の文章を英語に翻訳するためのプロンプトを生成します。直訳から段階的に自然な英語表現へと改善します。

  • Natural Phrasing Guide

    This tool provides suggestions for more natural phrasing in conversation transcripts, helping improve clarity and flow.

  • CLI Tool Prompt Generator

    This tool helps you generate command-line interface prompts for creating Python3 scripts with various specifications.

  • Sentence Refinement

    This tool reviews and refines sentences through iterative steps, providing suggestions to improve clarity and style.

  • Prompt Template Parser

    This tool converts extended Markdown with custom syntax into complete HTML prompt templates.
